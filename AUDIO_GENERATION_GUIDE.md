# 语音克隆音频生成指南

## 需求说明
需要保留克隆音频的音色音质，但内容要按自定义提示词朗读。

## 解决方案

### 方案1：使用AI语音克隆工具（推荐）

#### 工具推荐
1. **GPT-SoVITS**（开源免费）
   - 项目地址：https://github.com/RVC-Boss/GPT-SoVITS
   - 功能：5秒语音即可克隆，支持中文
   - 使用方法：
     ```bash
     # 1. 准备克隆音频（如：蜡笔小新.mp3）
     # 2. 训练声音模型
     # 3. 输入自定义文本生成新音频
     ```

2. **剪映专业版**（简单方便）
   - 打开剪映 → 文本朗读 → 选择"克隆音色"
   - 上传音频样本 → 输入自定义文本 → 导出

3. **讯飞配音**（在线服务）
   - 网址：https://peiyin.xunfei.cn
   - 支持声音克隆功能

#### 音频文件命名规则
将生成的音频放入 `audio/voices/` 目录：

```
audio/voices/
├── labixiaoxin/
│   ├── default.mp3          # 默认提示音
│   ├── 该写作业了.mp3       # 自定义内容1
│   ├── 该收拾玩具了.mp3     # 自定义内容2
│   └── ...
├── xiaowanzi/
│   ├── default.mp3
│   └── ...
└── ...
```

### 方案2：使用TTS+音效混合（当前实现）

由于浏览器限制，目前使用系统TTS语音合成，通过调整音调、语速来模拟不同角色。

**局限性：**
- 无法完全还原克隆音频的音色
- 只能调整语速、音调、音量

### 方案3：服务器端AI合成（高级）

搭建语音合成服务器，调用AI模型实时生成语音。

需要：
- Python + FastAPI服务器
- GPT-SoVITS模型
- GPU服务器

## 当前实现

目前使用浏览器内置TTS，通过参数调整模拟不同声音：

| 角色 | 语速 | 音调 | 特点 |
|------|------|------|------|
| 蜡笔小新 | 1.0 | 1.0 | 正常童声 |
| 樱桃小丸子 | 1.0 | 1.1 | 稍高童声 |
| 哪吒 | 1.0 | 0.9 | 稍低少年音 |
| 动漫小妲己 | 0.9 | 1.2 | 甜美高音 |
| 儿童 | 1.1 | 1.2 | 活泼高音 |
| 曼波女声 | 0.9 | 1.0 | 温柔中音 |
| TVB女声 | 1.0 | 1.0 | 标准女声 |

## 建议

1. **短期**：使用当前TTS方案，调整参数模拟不同声音
2. **中期**：使用剪映生成常用短语的音频文件
3. **长期**：接入AI语音合成API，实现真正的声音克隆
